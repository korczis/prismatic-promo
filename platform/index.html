Platform

What Prismatic Is

Prismatic Platform is an intelligence infrastructure. It is a runtime -- built on Elixir/OTP -- that ingests signals from external sources, resolves them into a knowledge graph, validates claims through a formal epistemic framework, and exposes the results through real-time dashboards and APIs.

The platform is not a single application. It is 93 OTP applications organized within a Phoenix umbrella, coordinated by 434 autonomous agents operating across 14 specialized domains. At approximately 2.8 million lines of code, it represents a convergence of open-source intelligence collection, external attack surface management, corporate due diligence automation, and AI-driven drift monitoring -- unified under a single deterministic, event-sourced runtime.

What This Is NOT

  • Not a SaaS product you can sign up for today. Prismatic is infrastructure, not a hosted service. Deployment requires Elixir runtime and PostgreSQL.
  • Not a general-purpose AI assistant. The platform uses LLMs for specific agent reasoning tasks, but it is not a chatbot or general Q&A system.
  • Not a replacement for human analysts. It provides structured, provenance-tracked evidence for humans to make better decisions. It does not make decisions autonomously.
  • Not a security product in isolation. EASM and security ratings are capabilities within a broader intelligence infrastructure, not standalone security tools.

Key Architecture Decisions

DecisionRationale
Elixir/OTPFault-tolerant concurrency, hot code reloading, and the BEAM VM's preemptive scheduler handle 434 concurrent agents without thread management
Umbrella architecture93 OTP apps provide strong boundary enforcement while enabling shared compilation and coordinated releases
Event sourcingEvery significant action is an immutable event, making the entire system reproducible, auditable, and temporally queryable
Polyglot storagePostgreSQL (relational), ETS (microsecond cache), Meilisearch (full-text search), KuzuDB (graph), Redis (distributed cache) -- each used for what it does best
AIAD agent standardDeclarative agent definitions ensure predictable, auditable, composable behavior across all 434 agents
