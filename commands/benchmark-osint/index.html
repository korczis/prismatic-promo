/benchmark-osint

Performance L2 Production medium usage

Comprehensive OSINT adapter performance validation with 3NL refined approach

Overview

/benchmark-osint is a production command in the Performance category of the Prismatic Platform. It provides systematic benchmarking of all 120 OSINT adapters across 7 intelligence categories using an enforced 7+/-2 test suites per adapter pattern based on Miller's Law cognitive optimization.

This command operates under the L2 authority level and is executed by the osint-benchmark-specialist agent. It generates 840 benchmark suites with P50/P95/P99 percentile analysis, budget compliance checking, and auto-opening HTML reports with interactive visualizations.

Usage

# Run all OSINT benchmarks
/benchmark-osint

# Quick validation (reduced iterations)
/benchmark-osint --quick

# Comprehensive analysis (extended iterations)
/benchmark-osint --comprehensive

# Specific intelligence category
/benchmark-osint --category threat_intelligence
/benchmark-osint --category czech_intelligence

# Generate Markdown and JSON reports
/benchmark-osint --report

# Mix task equivalent
mix benchmark.osint --category domain_intelligence --report
