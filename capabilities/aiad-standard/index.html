AIAD Standard

DOCTRINE

AI-Agent Interface Definitions for 400+ autonomous agents

Overview

AIAD (AI-Agent Interface Definition) is the standard format for defining autonomous agents in the Prismatic ecosystem. Every agent, every command, every workflow follows this specification.

Components

Agents

# .aiad/agents/archer-supreme.agent.md
agent-spec:
  name: "archer-supreme"
  version: "2.0.0"
  tier: "L5-Supreme"
  description: "Ultimate authority for impossible missions"

  capabilities:
    - multi-phase operations
    - crisis intervention
    - zero-error execution

  tools:
    - Read
    - Write
    - Edit
    - Bash
    - Task
    - WebFetch

  enforcement:
    doctrine: "no-mercy-no-doubts"
    compliance: mandatory

Commands

# .aiad/commands/orchestrate.cmd.md
command-spec:

  parameters:
