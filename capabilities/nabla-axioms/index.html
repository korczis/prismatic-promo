NABLA Axioms

DOCTRINE

Seven non-negotiable epistemic axioms governing all belief formation

Overview

NABLA (∇∞) defines seven axioms that govern how the platform reasons about uncertainty. These axioms are non-negotiable - every agent, every decision, every belief must comply.

The Seven Axioms

1. Signal Plurality

No belief without at least two independent signals.

RequirementEnforcement
Minimum 2 signalsHARD
Independent sourcesSOFT
Cross-validationHARD
# Violation: Single source
{:error, :insufficient_signals}

# Valid: Multiple sources
{:ok, belief} when length(signals) >= 2

2. Contradiction Preservation

Never discard contradicting evidence.

Both sides of a contradiction must be preserved and tracked. Resolution requires explicit reconciliation, not deletion.

3. Absence Is Informative

Missing signals carry meaning.

The absence of expected data is itself data. Track what you expected but didn't find.

4. Time Decay

All beliefs have mandatory timestamps.

Confidence decays over time. Recent signals weight more than old ones.

decay_factor = :math.exp(-age_hours / @half_life)
adjusted_confidence = base_confidence * decay_factor

5. Unknown Is Valid

"I don't know" is a valid answer.

Never fabricate certainty. Explicit uncertainty is always preferable to false confidence.

6. Source Independence

Independent sources weight higher.

Signals from the same source family get diminishing returns. True independence increases confidence.

7. Provenance Required

All beliefs must be traceable.

Every belief traces back to its originating signals. No orphan beliefs allowed.
