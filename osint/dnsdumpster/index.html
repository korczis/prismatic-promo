Overview

DNSdumpster is a free DNS reconnaissance and research tool created by HackerTarget.com. It discovers subdomains, DNS records, and host information for any domain using multiple data sources including search engine scraping, certificate transparency logs, and active DNS resolution. DNSdumpster is notable for its visual network mapping feature that generates graphical representations of a domain's DNS infrastructure.

For OSINT analysts, DNSdumpster provides a quick, free starting point for domain reconnaissance. While it lacks the depth of commercial tools like SecurityTrails, its visual output and zero-cost access make it valuable for initial domain mapping and quick infrastructure overviews.

Key Features

  • Automated subdomain discovery from multiple passive sources
  • Visual DNS infrastructure mapping with downloadable graphics
  • MX, NS, and A record enumeration with banner information
  • Geographic mapping of discovered hosts

API Access

DNSdumpster does not provide an official API. Access is through the web interface at dnsdumpster.com. HackerTarget.com offers a related API service with limited free queries (10/day) and paid plans for programmatic access. Results are available as structured data through the web interface and as visual PNG/SVG maps. The Prismatic adapter uses structured web interaction for automated data extraction.

Use Cases

  • Initial domain reconnaissance during the early stages of an investigation
  • Quick subdomain discovery for small to medium-sized domains
  • Visual infrastructure mapping for presentations and reports
  • Identifying email infrastructure through MX record analysis
  • Cross-validating subdomain findings from other tools

Integration

Prismatic Platform integrates DNSdumpster as a complementary domain reconnaissance source. Results are merged with SecurityTrails DNS data, crt.sh certificate transparency records, and Censys scan data to produce comprehensive subdomain inventories. DNSdumpster's visual mapping capability is used for automated report generation in the Prismatic Perimeter attack surface assessments.
