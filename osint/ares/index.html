ARES

czech API

Access Register of Economic Subjects - the central Czech business registry

Type: company
Module: Ares
Official Website

Capabilities

Company Search ICO Lookup DIC Validation Company Details Statutory Bodies Address Verification

Overview

ARES (Administrativní registr ekonomických subjektů) is the central information system of the Czech Republic that aggregates data from multiple public registers. It provides a unified interface to search for economic subjects registered in the Czech Republic.

Data Sources

ARES aggregates data from:

RegistryData Type
Obchodní rejstříkCommercial companies
Živnostenský rejstříkTrade licenses
RESStatistical register
DPHVAT payer registry
Registr plátců spotřební daněExcise duty payers

Available Data

Company Information

  • Legal name and trade name
  • ICO (Company ID) and DIC (Tax ID)
  • Legal form
  • Date of incorporation
  • Registered address
  • Business activities (NACE codes)

Statutory Bodies

  • Directors and board members
  • Shareholders (for some entities)
  • Authorized signatories

Financial Indicators

  • Capital structure
  • Number of employees (range)
  • Annual turnover category

Query Examples

# Search by company name
{:ok, results} = Ares.search(name: "Prismatic")

# Lookup by ICO
{:ok, company} = Ares.get_by_ico("12345678")

# Validate DIC
{:ok, valid?} = Ares.validate_dic("CZ12345678")

# Get full details
{:ok, details} = Ares.get_full_details("12345678")

Integration Tips

Rate Limiting

While ARES doesn't publish official rate limits, we recommend:

  • Maximum 1 request per second
  • Implement exponential backoff
  • Cache results for 24 hours

Error Handling

case Ares.get_by_ico(ico) do
  {:ok, company} -> process_company(company)
  {:error, :not_found} -> handle_missing()
  {:error, :rate_limited} -> retry_later()
  {:error, reason} -> log_error(reason)
end
