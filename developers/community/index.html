Contribution Workflow

# 1. Fork repository
git clone https://github.com/your-username/prismatic-sdk
cd prismatic-sdk

# 2. Create feature branch
git checkout -b feature/custom-adapter-example

# 3. Implement changes with tests
npm test
npm run lint
npm run type-check

# 4. Commit with conventional format
git commit -m "feat(adapters): add example custom threat feed adapter

- Implement CustomThreatFeed adapter with rate limiting
- Add comprehensive test coverage
- Include documentation and usage examples
- Benchmark performance: <100ms average response time"

# 5. Open pull request with:
# - Clear description of changes
# - Test coverage report
# - Performance benchmarks (if applicable)
# - Breaking change notes (if any)
