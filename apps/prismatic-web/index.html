Prismatic Web

LiveView dashboards with TailwindCSS and Flowbite components

567 files Frontend

Overview

Prismatic Web provides the main LiveView dashboard interface for the platform. Built with Phoenix LiveView, TailwindCSS, and Flowbite components for responsive, real-time web applications.

Key Features

LiveView Dashboards

  • Real-time data updates without JavaScript
  • Server-rendered reactive interfaces
  • Automatic state synchronization
  • WebSocket-based communication

Design System

  • TailwindCSS: Utility-first CSS framework
  • Flowbite: Professional component library
  • JetBrains Mono: Code and data display font
  • Dark/Light Mode: Automatic theme switching

Dashboard Modules

  • HAWKEYE: Visitor intelligence monitoring
  • Perimeter: EASM security dashboard
  • Quality: Platform health monitoring
  • Agents: AI agent management interface

Architecture

Phoenix LiveView → Component Tree → TailwindCSS → Real-time Updates

Component Structure

# Reusable dashboard components
<.dashboard_card title="Metrics" icon="chart">
  <.metric_display value={@count} label="Active Sessions" />
</.dashboard_card>

# Real-time data tables
<.data_table rows={@visitors} columns={@columns} live_update />

Styling Standards

TailwindCSS-First

  • ✅ REQUIRED: TailwindCSS utilities only
  • ❌ FORBIDDEN: Inline styles, custom CSS
  • Design System: Consistent color palette, spacing, typography

Responsive Design

  • Mobile-first approach
  • Breakpoint-based layouts
