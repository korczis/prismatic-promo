Prismatic Compliance

NIS2, GDPR, and ZKB compliance assessment with automated evidence collection

380 files Compliance

Overview

Prismatic Compliance provides automated compliance assessment against major European regulatory frameworks, including the NIS2 Directive (EU 2022/2555), GDPR (EU 2016/679), and the Czech ZKB (264/2025 Sb.). It generates compliance reports, tracks remediation progress, and collects evidence automatically from the platform's security and intelligence data.

Compliance is not a checkbox exercise. The platform continuously monitors compliance posture and integrates assessment results into the Prismatic Perimeter security ratings, ensuring that compliance gaps directly influence security scores.

Key Features

NIS2 Directive Compliance

The NIS2 Directive (Network and Information Security Directive 2) establishes cybersecurity requirements for essential and important entities across the EU.

# Assess NIS2 compliance for an organization
{:ok, assessment} = PrismaticCompliance.assess(:nis2, "example.com",
  entity_type: :essential,
  sector: :digital_infrastructure
)
# => %{
#   framework: :nis2,
#   overall_score: 72,
#   status: :partially_compliant,
#   categories: %{
#     risk_management: %{score: 85, status: :compliant},
#     incident_handling: %{score: 60, status: :partially_compliant},
#     supply_chain: %{score: 45, status: :non_compliant},
#     business_continuity: %{score: 90, status: :compliant},
#     encryption: %{score: 78, status: :partially_compliant},
#     access_control: %{score: 80, status: :compliant},
#     vulnerability_management: %{score: 65, status: :partially_compliant}
#   },
#   remediation_items: [...],
#   evidence: [...]
# }

# Get specific NIS2 control assessment
{:ok, control} = PrismaticCompliance.assess_control(:nis2, "Art.21.2.d",
  target: "example.com"
)

ZKB (Czech Cybersecurity Act)

The Czech ZKB (Zakon o kyberneticke bezpecnosti, 264/2025 Sb.) implements NIS2 into Czech national law with additional Czech-specific requirements.

# Assess ZKB compliance
{:ok, zkb} = PrismaticCompliance.assess(:zkb, "example.com",
  operator_type: :essential_service,
  sector: :energy
)
# => %{
#   framework: :zkb,
#   overall_score: 68,
#   czech_specific_controls: %{
#     nukib_registration: %{status: :compliant},
#     incident_reporting: %{status: :partially_compliant, deadline_hours: 24},
#     czech_language_docs: %{status: :compliant},
#     local_contact: %{status: :compliant}
#   }
# }

│   └── Storage        # Evidence archive management
├── Reporting

│   └── Export         # Regulatory submission format export
└── Remediation
    ├── Tracker        # Remediation item tracking
    ├── Prioritizer    # Risk-based prioritization
    └── Notifier       # Deadline and progress notifications
